# OP-ENCRYPTIONS-Security-Toolkit
A sleek, browser-based security toolkit for developers and privacy-focused users. OP ENCRYPTIONS offers essential cryptographic tools that run entirely locally—no data leaves your device. Built with a cyberpunk-inspired UI and powered by the Web Crypto API, it delivers strong, production-grade security with simplicity.


⚙️ Features

🔐 AES-256-GCM Encryption
Secure, military-grade encryption using PBKDF2 key derivation
Random IV + salt generation for strong protection
Encrypt & decrypt directly in-browser

🔑 Password Generator & Analyzer
Cryptographically secure password generation (window.crypto)
Custom length & character sets
Real-time entropy calculation & strength analysis

⬡ Hash Generator
Supports SHA-256, SHA-512, and SHA-1 (legacy)
Instant hashing for text and files
Hash verification (compare integrity)

🧩 JWT Decoder & Inspector
Decode and inspect JWT tokens
View header, payload, and signature
Automatic expiration (exp) checks

📦 Base64 Encoder/Decoder
Encode/decode text and files
URL-safe Base64 support (RFC 4648)

🛡️ Privacy First
100% client-side — no backend, no tracking
No data transmission — everything stays on your machine
Fully offline-capable
🎯 Why This Tool?

Instead of juggling multiple online tools (and risking sensitive data exposure), OP ENCRYPTIONS gives you a unified, secure workspace for everyday cryptographic operations.
